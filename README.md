## 🧠 Overview

**Operation Velvet Ace** is an AI-driven stealth simulation game where the player executes a high-stakes heist inside a luxury casino resort.

The player gathers intelligence, navigates a dynamic environment, and makes decisions that are evaluated in real-time by an AI engine. Every action has consequences—raising suspicion, triggering security systems, or unlocking new paths.

---

## 🎮 Gameplay

* Start in the **Casino**
* Explore the casino and restricted areas
* Collect intel and critical items
* Bypass security systems
* Access underground vault (B4)
* Steal the target asset
* Escape before **Heat reaches 100%**

---

## 🔥 Core Features

### 🧠 AI Game Master

* Uses LLM to evaluate player decisions
* Generates dynamic consequences

---

### 🗺️ Tactical Map System

* Zone-based navigation
* Real-time position tracking
* Structured movement system

---

### 📁 RAG-Based Intelligence System

* Game knowledge stored in structured files
* AI retrieves relevant context using vector search
* Ensures realistic and consistent gameplay

---

### 🚨 Heat System

* Tracks player risk level
* Increases with suspicious actions
* At 100% → mission failure

---

### 🕵️ Realistic Heist Mechanics

* CCTV looping
* Sensor disabling
* Keycard + biometric authentication
* HVAC infiltration paths
* Guard behavior patterns

---

## 🎯 Key Design Principles

* **Realism over shortcuts**
* **System-driven gameplay (not scripted)**
* **AI as a decision engine, not just text generator**
* **Consistent world logic via structured data**
