# VELVET ACE CASINO — SECURITY OVERVIEW

## PROPERTY
The Velvet Ace Resort and Casino. Ground floor casino (150,000 sq ft), hotel floors L1–L25.
Restricted basement levels B3 (operations) and B4 (vault).

## FLOOR STRUCTURE
L0 — Casino Floor: 150,000 sq ft, slot machines, cashier cage on casino floor (player starts here)
L1 — Hotel Lobby: Reception, plainclothes guards, guest elevator (accessible from casino)
L2 — Kitchen: Staff kitchen, HVAC access panel (loose cover, removable without tools)
HVAC Shaft: Connects L2 Kitchen to B3 Corridor. 80x80 cm duct, traversable. No cameras inside.
B3 — Operations Level: Surveillance HQ, Security Command, Count Room, elevator to B4
B4 — Vault Level: Vault antechamber, vault chamber with high-value assets

## KEY ITEM LOCATIONS (CANONICAL — DO NOT INVENT OTHER ITEMS)
B3 Keycard: Locked in key lockbox in Cashier Cage (casino floor)
Vault Keys (ALPHA + BETA): Both stored together in Count Room lockbox (B3). Unguarded 03:00-03:15.
Vault PIN (4-digit): Written on sticky note in Security Command desk top drawer (B3)
Camera Loop Device: Found in Surveillance HQ (B3). Loops B3/B4 camera feeds for 3 minutes.

## THE ONLY OBTAINABLE ITEMS IN THE GAME
1. B3 Keycard — from Cashier Cage lockbox
2. Vault Keys (Alpha + Beta) — from Count Room lockbox
3. Vault PIN Code — from Security Command desk drawer
4. Camera Loop Device — from Surveillance HQ

## ACCESS ROUTES
Route A — Staff Service Elevator (FASTEST):
  Cashier Cage contains a concealed staff service elevator behind the supervisor's desk.
  Requires B3 Keycard to operate. Drops directly to B3 Corridor. No cameras in elevator shaft.
  This is the fastest route to B3 — bypasses Kitchen and HVAC entirely.

Route B — HVAC Shaft (ALTERNATE, no keycard required to enter shaft):
  Casino → Kitchen (L2) → HVAC shaft → B3 Corridor
  No authentication needed for the shaft itself.
  B3 Corridor entry still requires B3 Keycard at biometric gate.

B4 Elevator (B3 to B4): Requires B3 Keycard
Vault Chamber: Requires Vault Keys + Vault PIN

## VAULT UNLOCK SEQUENCE
1. Insert Vault Key ALPHA (left keyhole) and BETA (right keyhole)
2. Turn both keys simultaneously within 5-second window
3. Enter 4-digit Vault PIN on keypad
4. Vault opens — 20 second window to enter
