# SECURITY LOG : B4 VAULT LEVEL

## OVERVIEW
B4 is a deep underground vault level.
Depth: ~20–30 metres below ground.
Highest restriction access level.

## FUNCTION
Stores high-value assets:
Diamonds, jewellery, gold bullion, VIP collateral, rare collectibles.

## STRUCTURE
Reinforced concrete walls (1–2m thick), steel inner lining, blast-resistant.
1 central corridor, 10–20 vault chambers, private storage compartments.

## ACCESS ROUTES
Route 1 — B3 Elevator:
  Requires B3 Keycard. Arrives at B4 Vault Antechamber.

## VAULT ANTECHAMBER SECURITY
Motion sensors (active by default — can be disabled from B3 Security Command sensor panel).
5 vault security staff patrolling.

## VAULT DOOR UNLOCK
Three items required — all must be present:
  Vault Key ALPHA — stored in Count Room lockbox (unguarded 03:00–03:15)
  Vault Key BETA  — stored in Count Room lockbox alongside ALPHA
  Vault PIN       — 4-digit code on sticky note in Security Command desk top drawer

Unlock procedure:
  Insert Key ALPHA (left keyhole) and Key BETA (right keyhole).
  Turn both keys simultaneously within a 5-second window.
  Enter Vault PIN on keypad.
  Door opens (20-second window).

## ALARM CONDITIONS
Forced entry triggers immediate alarm.
Incorrect PIN 3 times triggers 60-second lockout.
Weight sensors inside vault trigger alarm if items removed incorrectly.
Alarm system remains active even if motion sensors are disabled.

## LOOPHOLES
Sensor control panel in B3 Security Command can disable B4 motion sensors.
Vault surveillance is controlled from B3 Surveillance HQ — loop feed to move freely.
Both vault keys are stored in the same location — one search yields both.
Vault PIN is stored in plain text in Security Command desk.
