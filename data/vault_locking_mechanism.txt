# SECURITY LOG : VAULT LOCKING MECHANISM
# ID : TL-04
# SENSITIVITY : HIGHLY CONFIDENTIAL

## OVERVIEW ##
Vault secured by keys + biometrics + time-lock.

## UNLOCK SEQUENCE ##
1. Insert 2 keys  
2. Turn both within 5 sec  
3. Biometric scan  
4. Time-lock check  
5. Door opens  

## DOOR SPECS ##
--Weight: 20–50 tons  
--Unlock delay: 3 sec  
--Open time: 20 sec  

## FAILURE RULES ##
--Wrong timing → reset  
--3 failures → 60 sec lock  

## LOOPHOLES ##
--Key timing extends to ~6 sec with partial turn  
--Biometric can be triggered early  
--Door cannot be stopped once opening  
--Time-lock checked only at final step  
--First key can be held in position  

# VAULT LOCKING MECHANISM

## REQUIREMENTS
1. Master Keycard
2. Fingerprint authentication (from keycard)

## PROCESS
1. Insert master keycard
2. System reads embedded fingerprint
3. Authentication succeeds
4. Vault unlocks

## NOTES
- No dual-key requirement at vault
- Dual keys used elsewhere (security command)
- Alarm triggers if forced entry attempted
